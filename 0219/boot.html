02_부트스트랩(글자, 문단관련)

부트스트랩에서는 기본 글자크기가 16px이고, 행높이(줄간격)는 1.428이 기본값이다.

위 내용은 문단태그(p)요소에 자동으로 적용되며, 하단여백(margin)은 16px이 기본값이다.

h2 - 31.136px

h3 - 27.424px

h4 - 23.712px

h5 - 20px

제목요소인 h태그는 margin -bottom이 8px이 기본값이다.

small - 글자를 작게 설정 : 14px
mark - 글자에 하이라이트 효과를 적용함. 부트스트랩이 적용 아닐시 노란색 배경으로 나타남 (16px)
abbr - 약어나 줄임말을 사용하고자 할 떄 사용함 (16px)
blockquote - 다른 사람의 말을 대신하여 전하는 글 (글자크기 14px, 글자색상 #6c757d)

문단관련 서식

class = lead : 단락을 눈에 띄게하기(글자크기 : 20px)

class = small : 상위문자 크기의 85% = 14px로 문자크기를 작게 설정함

class = text-start : 문단정렬(왼쪽정렬)

class = text-end : 문단정렬(오른쪽정렬)

class = text-center : 문단정렬(가운데정렬)

class = text-justify : 문단정렬(양쪽배분정렬)

class = text-nowrap : 문단끝에서 글자가 다음줄로 넘어가지 않기

class = text-uppercase : 문단를 대문자로 출력

class = text-lowercase : 문단를 소문자로 출력

class = text-capitalize : 영문자의 첫글자만 대문자로 출력
